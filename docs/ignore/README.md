# OwnPay Documentation Index

This directory contains the documentation for the OwnPay project, organized for better discoverability.

## Directory Structure

- **[fintech/](fintech/)**: Documentation specific to the Fintech integration.
  - **[migration/](fintech/migration/)**: Task lists and plans for cutover and migration.
  - **[release/](fintech/release/)**: Release checklists and SOPs.
  - **[schema/](fintech/schema/)**: Database schema changes and impact guides.
- **[reports/](reports/)**: Audit and fix reports.
- **[plans/](plans/)**: Implementation and working plans.
- **[security/](security/)**: Security audits and finding reports.
- **[database/](database/)**: General database tasks and consolidation plans.
- **[reference/](reference/)**: Master reference docs and PR guidelines.
- **[architecture/](architecture/)**: High-level system architecture and extensibility plans.
- **[op/](op/)**: Core API and business logic documentation.
- **[mobile_app/](mobile_app/)**: Mobile application documentation.
- **[admin_panel/](admin_panel/)**: Admin panel specific documentation.
- **[plugins/](plugins/)**: Plugin system documentation.
- **[theme/](theme/)**: Theme development guides.
- **[checkout-theme/](checkout-theme/)**: Checkout theme documentation.
